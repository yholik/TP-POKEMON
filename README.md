# TP-POKEMON

Enciclopedia visual interactiva desarrollada para la materia **Aplicaciones Web** del Prof. Cesarini, Juan Pablo. El proyecto es una aplicación frontend nativa orientada al consumo dinámico de datos de Pokémon y la gestión interactiva de la interfaz de usuario en un entorno de desarrollo local.

## 🚀 Características y Tecnologías
* **Frontend Core:** Desarrollado con **HTML5** estructurado y **CSS3** personalizado para lograr una interfaz temática interactiva con efectos visuales adaptativos.
* **JavaScript Asincrónico (Módulos API):** Implementación de consumo de datos mediante scripts dedicados (`pokemon.js` y `pokemon_delay.js`) para gestionar las peticiones de la enciclopedia de forma asíncrona.
* **Manipulación del DOM:** Arquitectura limpia en la capa de presentación mediante scripts de interfaz (`ui.js` y `main.js`) para actualizar la vista de forma fluida y sin recargar la página.

## 🛠️ Cómo ejecutar el proyecto
El proyecto está configurado para ejecutarse localmente a través de un entorno de servidor web (como XAMPP):

1. Descargá o cloná este repositorio dentro de la carpeta raíz de tu servidor local (ej: `htdocs` en XAMPP o `www` en WampServer).
2. Asegurate de que el nombre de la carpeta se mantenga exactamente como `TP-POKEMON`.
3. Iniciá el módulo Apache desde el panel de control de tu servidor (XAMPP).
4. Abrí tu navegador web preferido e ingresá a la siguiente URL local:

```text
http://localhost/TP-POKEMON/
